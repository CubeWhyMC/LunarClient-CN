## LunarClient CN Loader

> 本仓库仅为Loader, 旧版本LunarCN已弃用, 其源代码可在其他分支中被找到

## 安装

> 我们推荐使用[Celestial](https://github.com/CubeWhyMC/celestial),
> 这是开箱即用的, 只需要在设置中打开LunarCN支持

1. 打开你的Lunar管理器
2. 下载LunarCN Loader
3. 将LunarCN Loader作为Agent进行添加
4. 启动游戏
